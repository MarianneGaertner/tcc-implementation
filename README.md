# Trabalho de Conclusão de Curso (TCC)
## PSO (Particle Swarm Optimization) em Ponto Fixo (MATLAB & VHDL)

Este repositório contém o desenvolvimento do algoritmo PSO aplicado à identificação de sistemas:

* **MATLAB**: Implementações do algoritmo utilizando aritmética de ponto flutuante e ponto fixo ($Q12.20$).
* **VHDL**: Descrição de hardware dos blocos da função custo (`COST_FUNCTION`), do multiplicador sequencial (`MULTIPLIER`) e de atualização da velocidade e posição das partículas (`PARTICLE_UPDATE`).
