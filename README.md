# Trabalho de Conclusão de Curso (TCC)
## PSO (Particle Swarm Optimization) em Ponto Fixo (MATLAB & VHDL)

Este repositório contém a implementação do algoritmo PSO utilizando aritmética de ponto flutuante e ponto fixo em Matlab e implementações dos blocos da função custo e atualização da velocidade e posição das partículas em VHDL.
